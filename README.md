# Indian-Food-logger
Diet tracking application with Material Design 3 (Material You).

Core Features Implemented:

    Dashboard: Daily calorie goal tracker, macro-nutrient breakdown ring.
    Diet Logging Flow: A highly intuitive search-and-add interface for meals, with a custom food logging toggle. The friction to log food is as low as MyFitnessPal. (all indian foods library adding in 2 days)
    Step tracking: coming soon.
